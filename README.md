<h1 align="center">Learn English</h1>

## Project Overview 🎉
it is a completely responsive application supporting the learning of English. Checks the level of skills and displays relevant questions, tasks and educational content
## Tech🔧
- React
- React Spring
- React router dom
- Styled Components
- Material UI
- Redux Toolkit
